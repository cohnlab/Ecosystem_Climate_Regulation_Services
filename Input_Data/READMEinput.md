
## Input_Data

This folder should have the following subfolders:

* `Avitabile`

* `CMIP6`

* `Crop_Calendar` containing NetCDF files in the same grid as the input climate data describing calendars for each crop, (ex. `Soybeans.crop.calendar.fill.nc`) 

* `Dias`

* `GIS`

* `GLOBIOM-BR`

* `Land_prices`

* `MAPBIOMAS`

* `Xavier`
