
## Input_Data

This folder should have the subfolders below. The folders contain pre-processed data from the references listed below under `References`.

* `Avitabile`
* `CMIP6`
* `Crop_Calendar` 
* `Dias`
* `GIS`
* `GLOBIOM-BR`
* `Land_prices`
* `MAPBIOMAS`
* `Xavier`
